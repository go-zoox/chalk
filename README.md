## Installation
To install the package, run:
```bash
go get github.com/go-zoox/chalk
```

## Getting Started

```go
import (
  "testing"
  "github.com/go-zoox/chalk"
)

func main() {
	fmt.Println(chalk.Red("Hello, World!"))
}
```
